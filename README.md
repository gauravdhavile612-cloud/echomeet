# 💫 EchoMeet

## Find Your People — Not Dating, Just Real Connections

EchoMeet is a personality-based friend-finding web app designed to help people build meaningful connections without the pressure of dating.

---

## ✨ Features

- 🧠 Personality-based matching  
- 🤝 Genuine friendships only  
- 🛡 Safe & friendly community  
- 🚫 No dating pressure  
- ⚡ Simple and clean interface  

---

## 🚀 Pages Included

- 🏠 Landing Page — `index.html`
- 📝 Signup Page — `signup.html`
- 🔍 Match Page — `match.html`

---

## 💻 Tech Used

- HTML  
- CSS  
- JavaScript (coming soon)

## 🌟 Vision

To become the world’s most trusted platform for real human connections beyond social media and dating apps.
